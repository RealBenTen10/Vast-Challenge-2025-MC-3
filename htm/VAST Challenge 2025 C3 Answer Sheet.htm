3 � It was noted by Clepper's intern that some people and vessels are using pseudonyms to communicate.

a.      Expanding upon your prior visual analytics, determine who is using pseudonyms to communicate, and what these pseudonyms are.

�       Some that Clepper has already identified include: Boss, and The Lookout, but there appear to be many more.

b.     Describe how your visualizations make it easier for Clepper to identify common entities in the knowledge graph.

Here are some of the Pseudonyms we were able to uncover:

  • Elise is using the pseudonyms Mrs. Money and The Accountant
  • Liam Thorne is using the pseudonym The Middleman
  • Rodriguez is using the pseudonym Small Fry
  • Sam is using the pseudonym The Intern
  • Kelly is using the pseudonym The Lookout
  • Davis is using the pseudonym V. Miesel Shipping
  • Sailor Shifts Team is using the pseudonym Glitters Team
  • Seawatch is using the pseudonym The Lookout
  • Serenity is using the pseudonym Remora
  • Nadia Conti is using the pseudonym Boss

Our visualizations enable the user to look through the Knowledge Graph using the Node-Link Diagram, Communication View, Bar Plot, Sankey and Event View. By Focusing on one Entity and using the Views together it is possible to find Pseudonyms by at the end looking at the Communication View. Since only the Communication content is 100% correct, the user has to decided whether it is a Pseudonym or not. Using the Similarity search it is possible to find Pseudonyms which are not directly visible otherwise (some Entities make sure that either A and B communicate or C and D with C being A's Pseudonym and D being B's Pseudonym). The Animation also helps in finding Pseudonyms since the user can sometimes see that after A and B communicated, C and D communicated. One easy example being Sam and Kelly with The Intern and The Lookout as Pseudonyms respectively. It is possible to find Pseudonyms through different means. One possibility is our Similarity Search.

Evidence of Liam using the Pseudonym "Middleman" and Elise using the Pseudonyms "The Accountant" and "Mrs. Money"

Another possibility is to find Pseudonyms through our Communication View directly by looking at the Messages in the correct time order.
